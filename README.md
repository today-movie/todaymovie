## 크롤링을 통한 코사인유사도 영화추천 시스템
- 사용자가 원하는 단어(ex 로맨스)를 입력하여 그에 맞는 영화를 추천해주는 시스템
- 왓챠피디아, 네이버영화 등 영화 리뷰 핵심 단어를 추출
- 추출한 데이터를 기반으로 입력값과의 코사인 유사도를 측정하여 추천
- 입력된 단어와 유사한 단어를 word2Vec을 사용하여 출력
- 추천된 영화를 기반으로 또 다른 유사한 영화를 제공

## 데이터 수집
- 영화진흥위원회 DB(20000101~20210930)
- 왓챠피디아 리뷰 DB(20000101~20210930)
- 네이버영화 리뷰 DB(20000101~20210930)

## 구현내용 요약
1. 웹 크롤링 과정
   - 파이썬 Beautiful Soup 라이브러리를 통해 웹 크롤링 진행
   - 크롤링된 리뷰 데이터 Mysql에 DB 저장
3. TF-IDF를 이용한 영화리뷰 코사인 유사도 분석
   - 불용어 제거, 명사/형용사/동사만 추출한 데이터 생성, 중복 제거(데이터 전처리)
   - 사이킷런 패키지를 이용한 TF-IDF를 통해 중요도가 높은 단어 가중치 부여 후 행렬 생성
   - TF-IDF 행렬을 통해 단어 벡터 간의 코사인 유사도 계산
   - 입력한 영화와 유사도가 높은 영화 추천 및 워드클라우드 생성
4. word2vec을 이용한 사용자 선택 키워드 기반 코사인 유사도 분석
   - gensim 패키지의 word2vec을 통해 워드임베딩 모델 생성
   - 키워드를 입력 시 영화리뷰 데이터를 토큰화 하여 워드임베딩 모델에서 유사한 단어들을 추출
   - 3개의 키워드의 각 유사한 단어들의 벡터를 합쳐 TF-IDF 모델을 통해 코사인 유사도 계산
   - 유사도가 높은 상위 영화 추출
5. 웹사이트 구현
   - 메인 : 로그인, 회원가입, 슬라이드 바를 사용하여 추후 배너 사용 가능)
   - 회원가입 : 사용자가 원하는 계정명과 비밀번호를 활용하여 생성)
   - 로그인 : 사용자가 생성한 계정으로 로그인)
   - 마이페이지 : 프로필 업데이트, 개인정보변경, 계정삭제
   - 영화 및 TV 프로그램 카테고리 : 각 카테고리 분류 목록 이동
   - 리뷰 카테고리 : 사용자가 직접 리뷰 작성 가능(게시글 작성 및 코멘트)
   - 찜 카테고리 : 사용자가 원하는 영화 및 TV 프로그램에 대해 찜 목록 추가
   - 스트리밍 사이트 연결화면
